            Since \(a\) and \(b\) are even, there exist integers \(k\) and \(m\) such that 
            \[
            a = 2k \quad \text{and} \quad b = 2m.
            \]
            Then,
            \[
            a+b = 2k + 2m = 2(k+m).
            \]
            Because \(k+m\) is an integer, \(a+b\) is divisible by 2 and therefore even.
        \end{lemma}
