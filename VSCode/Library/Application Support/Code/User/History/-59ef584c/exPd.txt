1. 请描述这两个命令的功能 ls -l .. 和 ls -ld ..
ls 是列出目录的意思, -l 表示 ‘long’ 的意思, 表示显示详细信息, 包括权限, 所有者, 大小, 修改日期等, -d 表示 ‘diretory’, 表示仅列出目录信息而不列出其内容, .. 表示上一级目录. 所以 ls -l .. 表示列出上一级目录下的内容, 以详细信息模式列出, ls -ld .. 表示列出上一级目录本身的详细信息, 不列出其中的内容.

3. 请描述这两个命令的功能：ls -lu  和 ls -lut
-u 是 "use” 的意思, 该选项本身不改变排列顺序, -lu 长形式下显示最近的访问时间, -t 是 “time” 的意思, 按照修改日期排序, 新修改的在前, -lut 的意思是列出长形式 (详细), 显示最近的访问时间, 且按照访问时间从新到旧排序.

5. 要将目录bar1拷贝到bar2，应该使用什么命令，目录bar2存在和不存在有什么区别？
cp -r bar1 bar2
若 bar2 存在, 则意思是递归地将整个目录 bar1 复制到 bar2/ 下, 新文件结构为 bar1/bar2, 若 bar2 不存在, 则意思是复制且重命名, bar2 和 bar1 在同一个父目录下, 除了名字外完全相同.
