1. 请描述这两个命令的功能 ls -l .. 和 ls -ld ..
ls 是列出目录的意思, -l 表示 ‘long’ 的意思, 表示显示详细信息, 包括权限, 所有者, 大小, 修改日期等, -d 表示 ‘diretory’, 表示仅列出目录信息而不列出其内容, .. 表示上一级目录. 所以 ls -l .. 表示列出上一级目录下的内容, 以详细信息模式列出, ls -ld .. 表示列出上一级目录本身的详细信息, 不列出其中的内容.
